JavaScript Loops Demo

This project demonstrates different types of loops in JavaScript using button click events.
When a user clicks a button, a specific loop executes and displays the output on the webpage.
The project helps in understanding how loops work and how JavaScript interacts with HTML using DOM manipulation.

# Features

The project includes the following loops:
✅ For Loop
✅ While Loop
✅ Do While Loop
✅ For...In Loop
✅ For...Of Loop
✅ Break & Continue
✅ Nested Loop

Each loop runs when its respective button is clicked.
